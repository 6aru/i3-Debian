# i3 Quick Start

Welcome to the JustAGuy Linux i3 setup. This file is yours to delete once you're comfortable.

## The one keybind to remember

**`Super + /`** — opens a searchable menu of every keybind. When in doubt, press it.

## Essentials

| Key | Action |
|---|---|
| `Super + Return` | Terminal (ghostty) |
| `Super + space` | App launcher (rofi) |
| `Super + b` | Browser |
| `Super + f` | File manager |
| `Super + e` | Text editor (geany) |
| `Super + q` | Close focused window |
| `Super + 1-0, -, =` | Switch to workspace 1-12 |
| `Super + Shift + 1-0, -, =` | Move window to workspace 1-12 |
| `Super + h/j/k/l` | Focus left/down/up/right |
| `Super + Shift + h/j/k/l` | Move window left/down/up/right |
| `Super + Shift + space` | Toggle floating |
| `Super + Shift + f` | Fullscreen |
| `Super + x` | Power menu |

## Layout

- **i3 config:** `~/.config/i3/config`
- **Keybinds (most):** `~/.config/i3/sxhkd/sxhkdrc` — edit and press `Super + Escape` to reload
- **Scripts:** `~/.config/i3/scripts/`
- **Theme switcher:** `~/.config/i3/scripts/thememenu` (12 themes available)
- **Theming:** `rofi/`, `polybar/`, `dunst/`, `picom/`
